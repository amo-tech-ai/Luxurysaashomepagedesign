## Real-World Example

**Dental AI Scheduling Tool**

**Score:** 72/100 â€” GO

**Section 1 - Problem:**
Dental practices lose $200 per unfilled appointment slot. Current systems require manual follow-up, leading to 20-30% no-show rates and frustrated staff.

**Section 3 - Market Sizing:**
- TAM: $2.1B (Global dental software market)

- **More Detail:** TAM validated through industry reports. SAM calculated by multiplying 68% SMB market share by US dental software spend.

- **Risks:** Single Statista source, no regional breakdown, growth extrapolated from 2021-2023.
- **Validate Next:** Survey 20 practices, verify pricing, interview DSOs.

### useReportContext Hook
```typescript
useReportContext(reportId: string) {
  const [context, setContext] = useState({})
  const [loading, setLoading] = useState(false)
  const cacheRef = useRef(new Map())

  const loadContext = async (section: number, content: string) => {
    if (cacheRef.current.has(section)) {
      return cacheRef.current.get(section)
    }

    const res = await fetch('/validator-panel-detail', {
      method: 'POST',
      body: JSON.stringify({ reportId, section, sectionContent: content })
    })
    const data = await res.json()
    cacheRef.current.set(section, data)
    setContext(prev => ({ ...prev, [section]: data }))
  }

  return { context, loading, loadContext }
}
```
