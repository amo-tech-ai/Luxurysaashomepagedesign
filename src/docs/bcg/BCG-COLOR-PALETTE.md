## Usage Guidelines

### Do's ✅

**BCG Green (#00D47E)**:
- ✅ Use for data visualizations (charts, graphs)
- ✅ Use for progress bars and indicators
- ✅ Use for secondary CTAs (outline buttons)
- ✅ Use for icons on light backgrounds
- ✅ Use for hover states on links

**Emerald Dark (#0E3E1B)**:
- ✅ Use for primary CTA buttons
- ✅ Use for navigation active states
- ✅ Use for important highlights
- ✅ Use in gradients with Sage

**Pastels (Mint/Sage)**:
- ✅ Use for card backgrounds
- ✅ Use for section backgrounds
- ✅ Use for subtle hover states
- ✅ Use in alternating patterns

### Don'ts ❌

**BCG Green (#00D47E)**:
- ❌ Don't use on pastel backgrounds (low contrast)
- ❌ Don't use for body text (readability issues)
- ❌ Don't use as primary button color (use Emerald)
- ❌ Don't overuse—reserve for data/accents

**Emerald Dark (#0E3E1B)**:
- ❌ Don't use for large text blocks
- ❌ Don't use on dark backgrounds
- ❌ Don't mix with BCG Green in same element

**Pastels**:
- ❌ Don't use white text on pastels
- ❌ Don't use pastels for CTAs
- ❌ Don't combine mint + sage in same card

## Accessibility Matrix

### Contrast Ratios (WCAG 2.1)

| Foreground | Background | Ratio | WCAG Level |
|------------|------------|-------|------------|
| #1A1A1A | #FFFFFF | 16.1:1 | AAA |
| #737373 | #FFFFFF | 4.6:1 | AA |
| #525252 | #FFFFFF | 8.4:1 | AAA |
| #FFFFFF | #0E3E1B | 13.8:1 | AAA |
| #FFFFFF | #1A1A1A | 16.1:1 | AAA |
| #00D47E | #1A1A1A | 3.2:1 | AA (Large text) |
| #0E3E1B | #DCF9E3 | 9.2:1 | AAA |
| #1A1A1A | #E8F5E9 | 15.1:1 | AAA |

**Legend**:
- **AAA**: Enhanced (7:1 for normal text, 4.5:1 for large)
- **AA**: Minimum (4.5:1 for normal text, 3:1 for large)
- **Large text**: 18px+ regular or 14px+ bold

## Color Applications

### Hero Section
```
Background:    #1A1A1A or Image + overlay
Text:          #FFFFFF

CTA:           #00D47E or #FFFFFF
```

### Content Cards
```
White Card:
- Background: #FFFFFF
- Border: #E5E5E5
- Text: #1A1A1A
- Icon: #00D47E or #0E3E1B

Mint Card:
- Background: #DCF9E3
- Text: #1A1A1A
- Icon: #0E3E1B

Sage Card:
- Background: #E8F5E9
- Text: #1A1A1A
- Icon: #0d5f4e

Dark Card:
- Background: #1A1A1A
- Text: #FFFFFF
- Icon: #00D47E
```

### Buttons
```
Primary Button:
- Background: #0E3E1B
- Text: #FFFFFF
- Hover: #0a2f14

Secondary Button:
- Background: Transparent
- Border: 2px solid #00D47E
- Text: #00D47E
- Hover bg: #E5F9F0

Text Link:
- Color: #00D47E
- Hover: #00B869
- Underline: Optional
```

### Data Visualizations
```
Bar Chart:
- Bars: #00D47E → #C1E8D0 (gradient)
- Background: #F5F5F5
- Labels: #525252
- Values: #1A1A1A
- Grid: #E5E5E5

Stat Display:
- Number: #00D47E or #0E3E1B

- Background: #FFFFFF or #DCF9E3
```

### Typography
```
Headlines:
- Color: #1A1A1A (light bg)
- Color: #FFFFFF (dark bg)
- Weight: 200-300 (ultra-light)

Body Text:
- Primary: #1A1A1A
- Secondary: #737373
- Tertiary: #A3A3A3

Links:
- Default: #00D47E
- Hover: #00B869
- Visited: #0d5f4e
```
