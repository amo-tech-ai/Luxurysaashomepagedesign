## Layout Structure

The Lean Canvas uses a **3-column responsive grid layout** with the following structure:

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              HEADER (Sticky)                                │
│  Background: #FFFFFF (80% opacity, backdrop blur)                          │
│  Border Bottom: #E5E7EB (gray-200)                                         │
│                                                                             │
│  [Logo: #3B5F52]  StartupAI        [Progress] [Save] [Download] [Share]   │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                           MAIN CONTENT AREA                                 │
│  Background: #FAF9F7 (cream/beige)                                         │
│  Max Width: 1600px                                                          │
│  Padding: 48px (desktop) / 24px (mobile)                                   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐  │
│  │  Page Title & Description                                           │  │
│  │  H1: #212427 (text-4xl lg:text-5xl font-light)                      │  │
│  │  Description: #6B7280 (gray-600, text-lg)                           │  │
│  └─────────────────────────────────────────────────────────────────────┘  │
│                                                                             │
│  ┌───────────────┬───────────────┬───────────────┐                        │
│  │   COLUMN 1    │   COLUMN 2    │   COLUMN 3    │                        │
│  │ Problem Side  │ Solution Side │ Customers &   │                        │
│  │               │               │   Economics   │                        │
│  └───────────────┴───────────────┴───────────────┘                        │
└─────────────────────────────────────────────────────────────────────────────┘
```

## Complete Color Palette

### **Background Colors**
| Element | Color Code | Name | Usage |
|---------|-----------|------|-------|
| Page Background | `#FAF9F7` | Cream/Beige | Main canvas background |
| Standard Card | `#FFFFFF` | White | All standard cards |
| UVP Card (Hero) | `#FBF9FA` | Warm Beige | Unique Value Proposition only |
| Input Field (Default) | `#F9FAFB` | Gray-50 | Textarea background (unfocused) |
| Input Field (Focus) | `#FFFFFF` | White | Textarea background (focused) |
| Header | `#FFFFFF` | White (80% opacity) | Sticky header with blur |

### **Border Colors**
| Element | Color Code | Name | Usage |
|---------|-----------|------|-------|
| Standard Card Border | `#E5E7EB` | Gray-200 | Default card borders (1px) |
| Card Border (Hover) | `#D1D5DB` | Gray-300 | Hover state |
| UVP Card Border | `#3B5F52` | Sage Green | Hero card border (2px) |
| Input Border (Default) | `#E5E7EB` | Gray-200 | Textarea border (1px) |
| Input Border (Focus) | `#3B5F52` | Sage Green | Textarea focus border (1px) |
| Header Border | `#E5E7EB` | Gray-200 | Bottom border (1px) |

### **Text Colors**
| Element | Color Code | Name | Usage |
|---------|-----------|------|-------|
| Headings (H1) | `#212427` | Charcoal | Page title |

## Design Principles

1. **Visual Hierarchy**: UVP card is visually elevated with unique background color and thicker border
2. **Generous Spacing**: 32px gaps between cards create breathing room
3. **Consistent Borders**: All borders use 1px except UVP (2px)
4. **Subtle Focus States**: Focus states use soft shadows and rings (not harsh outlines)
5. **Completion Indication**: Green checkmarks provide instant visual feedback
6. **Progress Tracking**: Header shows real-time completion percentage
7. **Muted Palette**: Sage green (#3B5F52) replaces bright emerald for sophistication
8. **Warm Neutrals**: #FBF9FA and #FAF9F7 create calm, premium feel
9. **High Contrast Text**: #212427 ensures readability on light backgrounds
10. **Accessible Interactions**: All interactive elements have clear hover/focus states

---

## Implementation Notes

- All colors use exact hex codes (no opacity variations except for rings/shadows)
- Focus rings use 10% opacity of sage green (#3B5F52)
- Card focus shadows use 5% opacity of sage green
- Header uses 80% opacity white with backdrop blur
- Checkmark icons use Lucide React's `Check` component
- All cards use `rounded-xl` (12px border radius)
- Input fields use `rounded-lg` (8px border radius)
- Transition durations: 300ms for cards, "all" for inputs
