## ðŸ“Š Understanding Slide Scores

Each slide is scored 0-100 based on:

- **Title Quality** (20 points): 10-60 characters optimal
- **Bullet Count** (15 points): 3-5 bullets ideal
- **Metrics Presence** (25 points): Numbers add credibility
- **Bullet Length** (20 points): 10-18 words per bullet
- **Speaker Notes** (10 points): Completeness
- **Specificity** (10 points): Avoid vague language

### Score Ranges
- **80-100**: Investor-ready âœ¨
- **60-79**: Good foundation, needs minor polish âš¡
- **0-59**: Needs improvement ðŸ”§

## ðŸ› Troubleshooting

### Deck Not Generating
- Check all required fields in wizard (marked with *)
- Wait for full 3-5 second generation time
- Refresh page if stuck

### Slides Not Saving
- Check for green "Saved" indicator
- Check browser console for errors
- Ensure localStorage is enabled

### Keyboard Shortcuts Not Working
- Don't use shortcuts while typing in text fields
- Check if another extension is capturing keys
- Refresh page to reset keyboard listeners

### Charts Not Displaying
- Ensure Recharts library is loaded
- Check browser console for errors
- Refresh page
